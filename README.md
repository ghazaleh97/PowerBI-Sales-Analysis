# Retail Sales Dashboard

## Project Overview
This project was developed using Microsoft Power BI to analyze retail sales data and create interactive business intelligence dashboards.

## Features
- Sales Trend Analysis
- Top Products Analysis
- Sales by City
- Category Filtering
- KPI Monitoring

## KPIs
- Total Sales
- Total Orders
- Total Quantity
- Average Sales

## Tools & Technologies
- Microsoft Power BI
- DAX
- Data Modeling
- Excel

## Concepts Used
- Calculated Columns
- Measures
- Top N Filtering
- Interactive Slicers
- Data Visualization
